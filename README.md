## Setup
- Clonar o repositorio:
  `git clone `https://github.com/dbguilherme/Sql-dojo.git

- Rodar o script: 
 `psql -h localhost -U postgres -d postgres -f script.sql `

- Acessar o terminal
  `sudo -u postgres psql postgres`
  \c dojo

- Fazer as questões do arquivo questions.sql :)
